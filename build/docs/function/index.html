    public findRootElement(node: DOMNode, selector: DOMSelector | undefined, Element: HTMLElement, defaultValue: Object | string | number): DOMNode | undefined source 

    import {findRootElement} from '@proddi/map-components/src/generics.js'

    Find a DOMElement. It has various ways:

    • try finding DOMNode by DOMSelector including a class check
    • iterate over parent-nodes to find a DOMNode of the specified class
    • find a DOMNode by querying by tag-name. Fails if not exactly one node found.
    • return default if specified or throw error.

    Params:

    NameTypeAttributeDescription
    node DOMNode

    The node to start lookup.
    selector DOMSelector | undefined

    The dom selector to find the element.
    Element HTMLElement

    The required type.
    defaultValue Object | string | number
    • optional

    Returned value if not found. NEED TO BE DIFFERENT THAN undefined!!

    Return:

    DOMNode | undefined

    The node or undefined if not found.

    Throw:

    *

    Exception when no {DOMNode} was found and no defaultValue was provided.
